# TMUX终端使用
一、Tmux 是什么？
1.1 会话与进程
命令行的典型使用方式是，打开一个终端窗口（terminal window，以下简称"窗口"），在里面输入命令。用户与计算机的这种临时的交互，称为一次"会话"（session） 。
会话的一个重要特点是，窗口与其中启动的进程是连在一起的。打开窗口，会话开始；关闭窗口，会话结束，会话内部的进程也会随之终止，不管有没有运行完。
一个典型的例子就是，SSH 登录远程计算机，打开一个远程窗口执行命令。这时，网络突然断线，再次登录的时候，是找不回上一次执行的命令的。因为上一次 SSH 会话已经终止了，里面的进程也随之消失了。
为了解决这个问题，会话与窗口可以"解绑"：窗口关闭时，会话并不终止，而是继续运行，等到以后需要的时候，再让会话"绑定"其他窗口。

1.2 Tmux 的作用
Tmux 就是会话与窗口的"解绑"工具，将它们彻底分离。
（1）它允许在单个窗口中，同时访问多个会话。这对于同时运行多个命令行程序很有用。
（2） 它可以让新窗口"接入"已经存在的会话。
（3）它允许每个会话有多个连接窗口，因此可以多人实时共享会话。
（4）它还支持窗口任意的垂直和水平拆分。
类似的终端复用器还有 GNU Screen。Tmux 与它功能相似，但是更易用，也更强大。

# TMUX 使用技巧整理
在 `tmux` 中，如果你想退出 `tmux` 会话，但不关闭或杀掉面板，以下是几种常见的做法：

### 1. **Detach 会话而不关闭面板**

你可以通过 **detach** （分离）会话的方式退出 `tmux`，这样 `tmux` 会话和面板将继续在后台运行，你可以稍后重新附加（reattach）到这个会话。

#### 使用快捷键：

- **`Ctrl+b d`**：分离当前会话，但会话中的所有窗口和面板将继续运行。

解释：

- `Ctrl+b`：进入 `tmux` 命令模式。
- `d`：分离当前会话，退出到普通终端，同时保留所有面板和会话状态。



在 `tmux` 中，可以通过键盘快捷键和命令切换窗口、面板等。以下是一些常用的命令和快捷键用于切换 `tmux` 的不同部分：

### 1. 切换窗口 (Window)

在 `tmux` 中，窗口类似于独立的终端会话。你可以通过以下方式在不同窗口之间切换：

#### 使用快捷键：

- `Ctrl+b n`：切换到下一个窗口。
- `Ctrl+b p`：切换到上一个窗口。
- `Ctrl+b 0-9`：直接跳转到指定编号的窗口。

#### 使用命令：

你也可以在 `tmux` 提示符下使用命令切换窗口：

- `Ctrl+b :select-window -t :2`：切换到编号为 2 的窗口。

### 2. 切换面板 (Pane)

一个窗口可以分割为多个面板。你可以通过以下方式在不同面板之间切换：

#### 使用快捷键：

- `Ctrl+b o`：切换到下一个面板。
- `Ctrl+b ;`：在最近使用的两个面板之间切换。
- `Ctrl+b {`：切换到上一个面板。
- `Ctrl+b }`：切换到下一个面板。

#### 使用方向键切换面板：

- ```
  Ctrl+b <方向键>
  ```

  ：根据方向键（上、下、左、右）切换面板。例如：

  - `Ctrl+b ↑`：切换到上一个面板。
  - `Ctrl+b ↓`：切换到下一个面板。
  - `Ctrl+b ←`：切换到左侧面板。
  - `Ctrl+b →`：切换到右侧面板。

#### 使用命令：

可以通过命令切换到特定的面板：

- `Ctrl+b :select-pane -t 1`：切换到编号为 1 的面板。

### 3. 切换会话 (Session)

`tmux` 允许同时运行多个会话，以下是切换不同会话的方法：

#### 使用快捷键：

- `Ctrl+b s`：显示所有会话，使用方向键选择并切换。
- `Ctrl+b (`：切换到上一个会话。
- `Ctrl+b )`：切换到下一个会话。

#### 使用命令：

- ```
  Ctrl+b :switch-client -t <session_name>
  ```
  ：切换到指定会话。例如：
  ```
  Ctrl+b :switch-client -t 0
  ```
以上这些快捷键和命令可以让你在 `tmux` 中高效地切换窗口、面板和会话。
# 窗口
#### 一、分割窗口（" %）
水平分割和垂直分割：
```
ctrl+b, "
ctrl+b, %
```
#### 二、面板切换（o 或方向键）
通常可以使用方向键来切换当前面板：
```
ctrl+b, 方向键
```
但手指离开了主键盘，总归不方便。如果不想离开主键盘，可以用 o 来做同样的事情：
```
ctrl+b, o
```

#### 三、全屏/恢复面板窗口（z）
用 z（zoom）来切换：
```
ctrl+b, z
```
#### 四、改变面板窗口大小（ctrl+方向键）
ctrl+b之后，然后按住ctrl不放，多次按方向键，就可以改变面板窗口大小。
```
ctrl+b, ctrl+方向键
```
复制到windows的剪切板：
```
按住shift键位
```
